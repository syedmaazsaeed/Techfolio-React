**🌌 Installation & Launch**

**Clone Repository:**

git clone https://github.com/syedmaazsaeed/Techfolio-React.git

**Navigate to Project Directory:**

cd portfolio

**Install Dependencies:**

npm install

**Launch the Website:**

npm start
